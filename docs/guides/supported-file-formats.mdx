title: Supported file formats
description: File formats supported by the Vertex platform.
---

Vertex supports the following file formats. We regularly add additional support. If you don't see your file format, [let us know](/contact).

| File type        | File extension                                   | Versions                                         |
| ---------------- | ------------------------------------------------ | ------------------------------------------------ |
| 3DXML            | .3DXML                                           | All                                              |
| CATIA            | .CATProduct\*, .CATPart, .model, .CATShape, .cgr | 4-6                                              |
| Creo/Pro-E       | .asm\*, .prt, .neu                               | Pro/Engineer 19.0-Creo 10                        |
| Creo View        | .pvz, .ol                                        | Creo View Express 4-7 and Creo View ECAD 6.1.0.0 |
| IFC              | .ifc                                             | 2-4                                              |
| IGES             | .iges, .igs                                      | Up to 5.3                                        |
| Inventor         | .iam\*, .ipt                                     | Up to 2025                                       |
| JT               | .jt\*\*                                          | 8.0-10.9                                         |
| NX - Unigraphics | .prt                                             | 11-12 and 1847-2312                              |
| OBJ              | .obj                                             | All                                              |
| Parasolid        | .x_b, .x_t, .xmt, .xmt_txt                       | Up to 36.1                                       |
| Revit            | .rvt, .rfa                                       | 2015-2024                                        |
| Rhino            | .3dm                                             | 4-8                                              |
| Solid Edge       | .asm\*, .par                                     | 19-20, ST1-ST10, and 2021-2024                   |
| SolidWorks       | .sldasm\*, .sldprt                               | 97-2024                                          |
| STEP             | .step, .stp                                      | AP203, AP214, AP242                              |
| STL              | .stl                                             | N/A                                              |

\* To import an entire assembly as a single file, you need to take an additional step to [prepare it for Vertex](https://github.com/Vertexvis/vertex-community/discussions/19#discussioncomment-660793).

\*\* To import the entire assembly as a single file for a shattered JT or NX assembly, [prepare it for Vertex](https://github.com/Vertexvis/vertex-community/discussions/19#discussioncomment-660793).
