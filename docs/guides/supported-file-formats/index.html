Supported file formats

Vertex supports the following file formats. We regularly add additional support. If you don't see your file format, let us know.

File typeFile extensionVersions
3DXML.3DXMLAll
CATIA.CATProduct*, .CATPart, .model, .CATShape, .cgr4-6
Creo/Pro-E.asm*, .prt, .neuPro/Engineer 19.0-Creo 7.0
IFC.ifc2-4
IGES.iges, .igsUp to 5.3
Inventor.iam*, .iptUp to 2021
JT.jt**8.0-10.5
NX - Unigraphics.prt11-12 and 1926
OBJ.objAll
Parasolid.x_b, .x_t, .xmt, .xmt_txtUp to 32
Revit.rvt, .rfa2015-2021
Solid Edge.asm*, .par19-20, ST1-ST10, and 2021
SolidWorks.sldasm*, .sldprt97-2021
STEP.step, .stpAP203, AP214, AP242
STL.stlN/A

* To import an entire assembly as a single file, you need to take an additional step to prepare it for Vertex.

** To import the entire assembly as a single file for a shattered JT or NX assembly, prepare it for Vertex.
